## 📖 Overview

This project provides a high-performance RESTful API designed for efficient management of student information. It offers a complete set of CRUD (Create, Read, Update, Delete) operations for student entities, making it suitable as a backend service for various applications requiring student data management. Built with Go and the lightweight Fiber web framework, it emphasizes speed, scalability, and ease of development. Data persistence is handled using sqlite and GORM, providing a reliable and structured database layer.

## ✨ Features

-   🎯 **Full CRUD Operations**: Create, retrieve, update, and delete student records via a RESTful interface.
-   ⚡ **High Performance**: Leverages the blazing-fast Fiber web framework for optimal API response times.
-   🗄️ **sqlite Database Integration**: Stores student data persistently in a robust sqlite database.

-   ⚙️ **Environment Configuration**: Easy setup and deployment using `.env` files for environment-specific settings.
